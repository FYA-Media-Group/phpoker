===> Spanish

Configure la dirección local y el puerto directamente en el archivo server.php
Configure algunas otras opciones de la mesa en settings.json
Ejecute server.php para crear una mesa

puede ejecutar varias veces server.php pero con diferentes puertos en la misma dirección local para crear más mesas

Cada instancia de server.php representa una mesa

El cliente debe ser subido a un servidor web y cargado correctamente usando la interface web
